# Assets Directory

This directory contains decorative motif images for the Ayurvedic theme.

## Required Assets

Place the following image files in this directory:

### Background Motifs
- **lotus-motif.png** - A subtle lotus pattern for page backgrounds
  - Recommended: Semi-transparent PNG, 200x200px or larger
  - Used as repeating background pattern

- **mandala-pattern.png** - A mandala pattern for section backgrounds
  - Recommended: Semi-transparent PNG, 300x300px or larger
  - Used for hero sections and card backgrounds

### Decorative Elements
- **corner-motif.svg** - Corner decorative element for cards
  - Recommended: SVG format for scalability
  - Used at card corners for elegant decoration

## Usage

These images are referenced in the CSS files:
- `styles.css` - Main stylesheet with background patterns
- `styles/remedies.css` - Remedy cards styling
- `styles/quiz.css` - Quiz page styling
- `styles/style_ritu-charya1.css` - Ritu Charya page styling

## Creating Your Own Assets

If you don't have these images, you can:
1. Search for "ayurvedic lotus pattern png" or "mandala pattern svg" 
2. Use design tools like Canva, Figma, or Adobe Illustrator
3. Ensure images are appropriately licensed for your use
4. Keep file sizes optimized for web (< 100KB recommended)
